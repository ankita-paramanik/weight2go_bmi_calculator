# Weigh2Go – BMI Calculator

Weigh2Go is a Python GUI application built using wxPython that calculates Body Mass Index (BMI) and visualizes the result on a graph with basic health tips.

## Features

- User-friendly graphical interface
- BMI calculation based on height and weight
- Graph visualization of BMI category
- Personalized health tips
- Gender selection and input validation

## Tech Stack

- Python
- wxPython
- Matplotlib

## How to Run

1. Install required libraries:-
pip install wxPython matplotlib

2. Run the program
python prog.py
